# Online Attendance System

## 📌 Project Description
This is a web-based Online Attendance System developed using Django. It helps teachers manage student attendance digitally.

## 🚀 Features
- Add Students
- View Students
- Mark Attendance
- Attendance Reports
- Dashboard Overview

## 🛠 Technologies Used
- Python (Django)
- HTML, CSS
- SQLite

## 🎯 Purpose
To automate attendance management and reduce manual work.
